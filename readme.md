### Requirements

- Node.js >= 12 (Node.js >= 14 is recommended)
- Environment variables specified in [.env.example](https://github.com/night-watch-project/spider-ham/blob/master/.env.example)

### Instructions

```shell
$ npm i # yarn install
$ npm start:dev # yarn start:dev
```
